Tentang Saya

Halo, nama saya ahmad arifin. Teman-teman saya biasanya memanggil saya arifin. Saya lahir di kota trenggalek jawa timur indonesia pada tanggal 14 juli 1999. saya sekarang sedang kuliah di salah satu kampus yang ada di malang jawa timur indonesia yaitu di institut asia malang jurusan teknik informatika.

Pada jurusan teknik informatika terdapat mata kuliah client side programming. Pada mata kuliah ini kita diajarkan untuk membuat tampilan suatu website menjadi menarik dan enak untuk dipandang.
